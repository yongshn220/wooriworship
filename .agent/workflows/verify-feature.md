description: Run automated browser testing for a newly implemented feature.
---

1.  **Read Context**: Read `test_credentials.md` and `test_workflow.md` to understand the environment.
2.  **Start Server**: Ensure the development server is running via `npm run dev` and capture logs.
3.  **Browser Setup**:
    - Navigate to `http://localhost:3000`.
    - Login using the credentials in `test_credentials.md`.
4.  **Execute Test**: Perform the specific actions described by the user to test the new feature.
5.  **Verification**:
    - Capture screenshots of the result.
    - Check terminal logs for any 500 errors or exceptions.
6.  **Report**: Provide a summary of the test results, including screenshots and log excerpts.
